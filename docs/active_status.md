# Active Build Status

**Updated:** 2026-07-21
**Controlling specification:** [project_brief.md](project_brief.md)

## Playable now

- One complete Ashwick Manor mystery with eight validated CCv3 characters.
- Authoritative discovery, investigation, interview, evidence, notebook, accusation, timeout, result, and debrief loops.
- Deterministic ten-minute turns and seven-NPC batch resolution from an immutable NPC-phase snapshot.
- Optional provider-neutral AI portrayal and intent selection. Every remote output is schema constrained and can only reference engine-authorized facts or finite action IDs.
- Versioned local saves, truth-redacted APIs, responsive desktop/mobile UI, and distinct portrait placeholders.
- Automated unit, contract, adversarial, persistence, AI-boundary, and black-box solve coverage, plus manual desktop/mobile browser passes.

## Still to build

1. Meaningful structured case assembly with at least a second authored crime spine and proof path. Seed-only movement or cosmetic evidence relocation will not be presented as replayability.
2. JSON CCv3 import/export plus a validation/editor workflow for user-supplied cards.
3. Additional expression artwork and richer deterministic emotional, suspicion, and private-conversation evolution.
4. Packaging and distribution verification after the game/content surface is stable.

The original continuous-real-time prototype documents are historical references. They do not describe the active turn-based build.
